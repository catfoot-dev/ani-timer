# AniTimer

![AniTimer](/public/logo.png)

애니메이션 타이머

## 개요

이 프로젝트에서는 애니메이터를 위한 애니메이션 타임 체크 도구를 만듭니다.

## 주요 기능

- 타이머 기능

## 기술 스택

- 프론트엔드: React

## 설치 방법

```sh
npm install
```

## 실행 방법

```sh
npm start
```
